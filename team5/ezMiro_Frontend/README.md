# frontend

## Environment
* Node.js: v15.11.0
* NPM: v7.6.0
* yarn: v1.22.10
* @vue/cli: v4.5.12

## Requirement Install
```
# Install Yarn
npm install -g yarn

# Install vue
yarn global add @vue/cli
```

## Build Setup
```
# Install Dependencies
$ yarn install

# Using the GUI
$ vue ui
```
